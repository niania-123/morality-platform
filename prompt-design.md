提示词设计文档：AI 课件生成工具

1. 我的原始需求
我想做一个网站，电脑端手机端都可以用，配备一个小程序，网页和小程序的风格是童趣水彩画风格。可以生成课件PPT，内容是小学一年级到六年级的道德与法治课，要符合当前年龄段学生的身心发展状态，符合新课改，ppt画风是水彩画风格，功能有以下几点：已自备一年级到六年级的道德与法治课ppt。支持提供其他道德与法治课的选题。支持通过选题或文字描述生成新的课件ppt。如果你有其他好想法可以问我或者跟我说。

2. 需求拆解与优化过程
| 原始需求 | 问题 | 优化后 |
| 生成课件PPT | 没有角色身份 | 你是一名小学道德与法治课专家 |
| 符合身心发展 | 规则不具体 | 一年级多用图片，六年级可加思辨 |
| 水彩画风格 | 没有落到输出 | 每页建议图片风格：水彩/童趣插画 |

3. 最终用于开发的 Prompt

 第一步：明确用户角色
原始 Prompt：没有指定 AI 的角色身份。  
优化后：你是一名小学道德与法治课专家，擅长设计符合 7-12 岁学生认知的课件。

第二步：明确输出格式
原始 Prompt：只说“生成课件PPT”。  
优化后：指定输出为 JSON 格式，包含封面、学习目标、正文、互动、总结等固定结构。

 第三步：加入年龄适配规则
原始 Prompt： 提到“符合身心发展状态”。  
优化后：
一年级：多用图片、短句、游戏化
三年级：加入文字段落、简单案例分析
六年级：可引入思辨讨论、价值观引导

第四步：加入风格约束
原始 Prompt“提到“水彩画风格”。  
优化后：在 Prompt 中加入画风描述，并在生成配图时调用 Pixabay API 匹配水彩/手绘风格图片。

最终用于开发的 Prompt 核心片段
markdown
你是一名小学道德与法治课专家。
任务：根据以下主题，生成一份小学课件PPT的内容大纲。

主题：{topic}
年级：{grade}（一年级到六年级）
页数：{pages}（默认12页）

要求：
1. 内容符合新课标，贴近学生生活
2. 语言难度适配该年级学生
3. 每页包含：类型（cover/intro/content/discussion/activity/summary）、标题、正文
4. 至少包含一个互动提问环节
5. 图片风格建议：水彩画/童趣插画

输出格式：JSON
